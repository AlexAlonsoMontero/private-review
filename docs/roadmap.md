# Roadmap

## Fase 1: Diseno y planificacion
- Definir MVP, modelo de datos y arquitectura.
- Prototipo de flujos clave.

## Fase 2: Desarrollo MVP
- Backend: grupos, resenas, lugares, reportes.
- Frontend web y movil basico.
- Login social y notificaciones push.

## Fase 3: Pruebas y validacion
- Pruebas unitarias e integracion.
- Pruebas E2E en web y movil.

## Fase 4: Despliegue inicial
- CI/CD con GitHub Actions.
- Entorno de produccion inicial.

## Fase 5: Evolucion y escalado
- Mejoras UX.
- Analitica avanzada.
- Monetizacion opcional.
