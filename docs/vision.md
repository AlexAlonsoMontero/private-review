# Vision del producto

## Problema
Las recomendaciones publicas no siempre reflejan el gusto real de una persona. La gente confia mas en su circulo cercano, pero hoy no existe una forma sencilla y privada de compartir resenas de lugares y servicios con ese circulo.

## Propuesta de valor
App de resenas privadas por grupos. Cada resena es visible solo para los miembros del grupo donde se publica. El objetivo es ofrecer confianza, privacidad y relevancia.

## Publico objetivo
Adultos de 20 a 70 anos que valoran la opinion de su circulo cercano para decidir donde comer, comprar o contratar servicios.

## Alcance inicial
- Grupos privados con al menos dos miembros.
- Resenas asociadas a lugares (fisicos u online).
- Busqueda de lugares y consulta de resenas solo dentro del grupo.
- Web y movil, con enfoque multiplataforma.

## Diferenciadores
- Privacidad real: las resenas no son publicas.
- Confianza: las opiniones provienen de personas conocidas.
- Moderacion privada: solo contenido reportado se revisa.

## Principios de producto
- Privacidad por defecto.
- Sencillez para crear grupos e invitar.
- Experiencia rapida para publicar y consultar resenas.

## Monetizacion (futuro)
- No hay anuncios ni venta de datos en el MVP.
- El sistema queda preparado para anuncios sutiles o datos agregados anonimizados, cumpliendo normativa.
