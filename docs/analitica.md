# Analitica y datos

## Objetivo
Medir uso y valor del producto sin exponer datos personales.

## Eventos base
- alta_usuario
- crear_grupo
- invitar_miembro
- aceptar_invitacion
- crear_resena
- ver_resena
- buscar_lugar
- reportar_contenido

## Metricas clave
- Usuarios activos diarios y mensuales.
- Tasa de conversion invitacion a miembro.
- Resenas por grupo.
- Crecimiento de grupos.
- Lugares mas resenados (anonimizado).

## Politica de datos
- Analisis sobre datos agregados.
- Sin exportar PII.
- Retencion limitada por necesidad de producto.
